## dict에서 zip함수 이용방법

```python
lux2 = dict(zip(['health','mana','melee','armor'],[490,334,550,18.72]))
lux2
# 출력: {'health':490, 'mana':334, 'melee':550, 'armor':18.72}
```

```python
# dict과 tuple로 딕셔너리 만들기
lux3 = dict([('health',490),('mana',334), ('melee',550),('armor',18.72)])
lux3
# 출력:{'health': 490, 'mana': 334, 'melee': 550, 'armor': 18.72}
```
