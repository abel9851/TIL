## format으로 문자열 정렬하기

```python
name='heejun'
a = '{name:>10}'.format(name=name) # :< 이면 문자열을 왼쪽으로 정렬하고 나머지는 공백으로 채움
								# :> 이면 문자열을 오른쪽으로 정렬하고 나머지는 공백으로 채움
print(a) # "    heejun"
```

## 숫자 개수 맞추기

```python
# '%0개수d' % 숫자
# '{인덱스:0개수d}'.format(숫자)
print('%03d'% 1) # 출력: 001
print('{0:03d}'.format(1)) # 출력: 001

# '%0개수.자릿수f' % 숫자 '.'도 8자리 중 한자리를 차지하기 때문에 0은 5개.
# '{인덱스:0개수.자릿수f}'.format(숫자)
print('%08.2f'% 3.6) # 출력: 00003.60
print('{0:08.2f}'.format(3.6))  # 출력: 00003.60
```
