## \***\*6020 : [기초-입출력] 주민번호 입력받아 형태 바꿔 출력하기(py)\*\***

입력:

주민번호 앞 6자리와 뒷 7자리가 '-'로 구분되어 입력된다.

(입력값은 가상의 주민번호이다.)

ex)110011-0000000

출력:

'-'를 제외한 주민번호 13자리를 모두 붙여 출력한다.

참고

아무것도 없는 공(empty) 문자는 작은 따옴표(') 2개를 붙여서 '' 로 표현한다.

입력예시: 000907-1121112

출력예시: 0009071121112

```python
# 풀이

a,b= input().split("-")
print(a,b,sep="")
```
