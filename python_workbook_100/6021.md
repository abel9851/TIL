## ****6021 : [기초-입출력] 단어 1개 입력받아 나누어 출력하기(설명)(py)****

알파벳과 숫자로 이루어진 단어 1개가 입력된다.

입력받은 단어의 각 문자를 한줄에 한 문자씩 분리해 출력한다.

입력: 5개의 문자로 이루어진 단어 1개가 입력된다.

출력: 각 문자를 한 줄에 한 문자씩 줄을 바꿔 출력한다.

입력예시: Hello

출력예시: 

H
e
l
l
o

```python
# 풀이1

s = input()
print(s[0])
print(s[1])
print(s[2])
print(s[3])
print(s[4])

# 풀이2
s = input()
print(s[0],s[1],s[2],s[3],s[4],sep="\n")
```
